# atrix-music-player
Atrix Music Player – پخش‌کننده موسیقی مدرن و سبک برای دسکتاپ و وب یک پلیر موسیقی سریع، زیبا و کاربرپسند که از انواع فرمت‌های صوتی پشتیبانی می‌کند، دارای رابط کاربری ساده، فهرست پخش قابل شخصی‌سازی و امکانات مدیریت کتابخانه موسیقی. ایده‌آل برای کاربران و توسعه‌دهندگانی که به دنبال یک پلیر سبک و قابل توسعه هستند.
