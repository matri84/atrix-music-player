# Atrix-music-player:
این یک برنامه تحت وب است که روی مرورگر شما اجرا میشود و شما با استفاده از آن میتوانید موزیک های خود را در محیطی شیک و بدون هیچ مشکلی اجرا کنید. این اپلیکیشن هیچ نیازی به اینترنت ندارد و به صورت لوکال روی سیستم شما اجرا میشود.
در ادامه شما یاد خواهید گرفت که چگونه از آن استفاده کنید و موزیک های خود را به آن اضافه کنید.
امید وارم لذت ببرید.
